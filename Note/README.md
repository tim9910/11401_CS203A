# C/C++ 程式

本資料夾用於存放本課程中與 **資料結構 (Data Structures)** 相關的 C/C++ 程式。  
---

## 1. 動態陣列記憶體配置 - `note1.cpp`

**重點說明：**
- 使用 `malloc()` 配置動態記憶體
- 使用 `realloc()` 擴充陣列大小
- 使用 `free()` 釋放記憶體
- 說明動態陣列需要 **連續記憶體空間**
- `realloc` 可能搬移舊資料到新位置，因此要重新檢查指標是否為 `NULL`

---

## 2. 鏈結串列建立與遍歷 - `note2.cpp`

**重點說明：**
- 使用 `struct Node` 建立**單向鏈結串列**
- 使用 `malloc()` 建立節點
- 串接方式： `head -> next -> ... -> NULL`
- 使用 `while` 走訪鏈結串列
- 最後逐節點 `free()` 回收記憶體
